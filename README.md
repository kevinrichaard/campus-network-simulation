# Campus Network Simulation using Cisco Packet Tracer

## Overview
This project demonstrates the design and implementation of a multi-department campus network using Cisco Packet Tracer. The network is structured to simulate real-world enterprise networking environments.

## Features
- Multi-department network architecture
- Routing using OSPF and RIP protocols
- Dynamic IP allocation using DHCP
- File transfer setup using FTP
- Inter-department communication

## Tools Used
- Cisco Packet Tracer

## File
- campus-network-simulation.pkt

## Learning Outcome
This project helped in understanding routing protocols, network configuration, IP management, and real-world network design concepts.
